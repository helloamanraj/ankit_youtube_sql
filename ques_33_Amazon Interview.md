Solution:
```sql

WITH cte AS (

  SELECT *,
    RANK() OVER (PARTITION BY dep_id ORDER BY salary DESC) AS max_rank,
    count(*) over (partition by dep_id ) as cnt
  FROM emp
 

)

select * from cte
where max_rank = 3 or (cnt < 3 and max_rank=cnt)
```
